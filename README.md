## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- Google Cloud Project with Gemini API
- Supabase project
- Firebase project

### Installation
1. Clone and setup frontend
2. Clone and setup backend
3. Configure environment variables
4. Run both services
